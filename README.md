# 1. Собрать docker-compose

```
make build
```

# 2. Поднять контейнеры

```
make up
```

# 3. Собрать, а затем поднять

```
make all
```

# 4. Посмотреть список образов и работающих контейнеров

```
make list
```

> К сожалению, пока возможно только получать кол-во записей в базе данных
